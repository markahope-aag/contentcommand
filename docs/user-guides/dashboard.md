# Dashboard Overview

The Content Command dashboard is your central hub for managing content creation, client relationships, and competitive intelligence. This guide covers all the main features and navigation.

## Dashboard Layout

### Navigation Sidebar

The left sidebar provides quick access to all major sections:

- **Dashboard** - Overview and key metrics
- **Content** - Content briefs, generation, and management
- **Clients** - Client management and settings
- **Analytics** - Performance metrics and reporting
- **Integrations** - API connections and health monitoring
- **Settings** - Organization and user preferences

### Top Navigation

The top bar includes:
- **Organization Switcher** - Switch between organizations (if member of multiple)
- **User Menu** - Profile settings and sign out
- **Notifications** - System alerts and updates

## Dashboard Home

### Key Metrics Overview

The dashboard home displays critical performance indicators:

#### Content Pipeline Status
- **Briefs in Progress** - Active content briefs
- **Content Generated** - Recently completed content
- **Quality Scores** - Average content quality metrics
- **AI Usage** - Current month's AI API consumption

#### Client Performance
- **Active Clients** - Number of clients with ongoing projects
- **Top Performers** - Clients with highest content performance
- **Recent Activity** - Latest client interactions and updates

#### Integration Health
- **API Status** - Health of connected integrations
- **Sync Status** - Last successful data synchronization
- **Rate Limits** - Current API usage vs. limits

### Quick Actions

Access frequently used features directly from the dashboard:

- **Create Content Brief** - Start new content planning
- **Generate Content** - Begin AI content creation
- **Add Client** - Onboard new client
- **View Analytics** - Access detailed performance reports

## Content Management

### Content Briefs

Content briefs are the foundation of your content strategy:

#### Creating a Brief
1. Click **Content** → **Briefs** → **Create Brief**
2. Enter target keyword and select client
3. AI analyzes competitors and suggests strategy
4. Review and approve the generated brief

#### Brief Components
- **Competitive Analysis** - SERP analysis and competitor insights
- **Content Strategy** - Unique angle and positioning
- **SEO Requirements** - Keyword optimization guidelines
- **AI Citation Opportunities** - Potential for AI platform visibility

### Content Generation

Transform approved briefs into high-quality content:

#### Generation Process
1. Select approved brief
2. Choose AI model (GPT-4, Claude, etc.)
3. Configure generation parameters
4. Review and edit generated content
5. Publish or export final content

#### Quality Control
- **Automated Scoring** - AI-powered quality assessment
- **SEO Analysis** - Keyword optimization and readability
- **Citation Readiness** - Optimization for AI platforms
- **Human Review** - Manual approval workflow

### Content Queue

Monitor all content in various stages:

#### Status Types
- **Draft** - Initial brief creation
- **Approved** - Ready for generation
- **Generating** - AI content creation in progress
- **Generated** - Content created, pending review
- **Reviewing** - Under human review
- **Revision Requested** - Needs modifications
- **Published** - Final approved content

#### Bulk Operations
- Filter by status, client, or date range
- Bulk approve or reject content
- Export multiple pieces of content
- Schedule content publication
